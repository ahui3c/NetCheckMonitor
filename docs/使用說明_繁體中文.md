# 對外網路連線能力監控程式 NetCheckMonitor 使用說明（繁體中文）

版本：0.9.1

NetCheckMonitor 可定時監控對外網路連線，記錄斷線並產生圖文報表，支援 Google Drive 備份與 PDF 下載。程式完全免費、開源、無廣告。

## 介面語言

- Windows 顯示語言為繁體中文（台灣、香港、澳門或 `zh-Hant`）時，使用繁體中文介面。
- 簡體中文及其他未支援語言預設使用英文。
- 不需要手動切換或另外下載語言檔。

## 開始監控

1. 執行 `NetCheckMonitor.exe`。
2. 預設每 60 秒檢查一次，可在開始前修改間隔。
3. 按「開始監控」。最小化視窗後仍會在系統匣持續執行。
4. 暫時不想納入測試時按「暫停」；恢復時按「繼續」。暫停區段會留下標示，但不計入統計。
5. 按「結束並產生報表」建立最終 HTML 報表。

## 即時報表與 PDF

- 監控期間可隨時按「產生即時報表」，檢查不會中斷。
- 程式每 10 分鐘自動更新 `_Live.html` 即時報表。
- 「下載報表 PDF 文件」可選擇全部資料或指定起訖日期。
- PDF 包含每日有效監控時間、每日估計斷線時間、斷線百分比、事件表與 24 小時時間軸。
- PDF 使用 Windows 內建 Microsoft Edge 的無介面列印功能。

## Google Drive 每日備份

1. 按「Google Drive 備份設定」。
2. 按「登入 Google Drive」，在系統瀏覽器登入自己的帳號並同意權限。
3. 設定並儲存每日備份時間；預設為 23:55。
4. 第一次連接會在 Drive 建立或沿用 `Net_Check` 資料夾。
5. 每日上傳完整 PDF 與原始 CSV；同名檔案會更新，不會一直產生重複檔案。

程式只要求 `drive.file` 權限。登入權杖由 Windows DPAPI 加密保存在目前 Windows 帳號中。程式必須保持執行才能準時備份；錯過時會在下次啟動優先補傳最近一天。

## 輸出檔案與識別碼

- 主要紀錄位於程式旁的 `NetCheck_Data`。
- 無法寫入程式目錄時，改存到「文件\NetCheck_Data」。
- 復原副本位於 `%LOCALAPPDATA%\NetCheck\Recovery`。
- 檔名使用 `NetCheck_電腦名稱-8碼識別碼_日期時間`。
- 8 碼識別碼優先由 Windows MachineGuid 單向雜湊產生，讀不到時才使用 MAC 位址或電腦名稱雜湊。
- 原始 MachineGuid、MAC 位址與硬體序號不會寫入 CSV、報表或檔名。

## 長時間執行與資料保護

- 每筆結果立即強制寫入主要 CSV 與復原副本。
- 監控中按視窗右上角 X 或 Alt+F4 只會縮到系統匣。
- 按右下角「關閉程式」時，會先停止監控、保存資料並建立最終報表。
- 監控時會要求 Windows 不要自動睡眠，但螢幕仍可依系統設定關閉。
- 筆記型電腦建議接上電源；闔上螢幕若設定為睡眠，Windows 仍可能進入睡眠。
- 請勿在監控期間直接拔除存放程式的 USB 隨身碟。

## 清除資料

主畫面右下角的「清除儲存資料」會刪除 NetCheckMonitor 管理的 CSV、HTML、即時報表及復原副本。下載到其他資料夾的 PDF 不會被刪除。監控或雲端備份進行中時，清除功能會取消以保護資料。

## 斷線百分比計算

每日斷線百分比＝該日估計斷線時間 ÷ 該日有效監控時間 × 100%。暫停時段會從兩者排除。斷線的開始與恢復時間是依相鄰檢查推估，精度受檢查間隔影響。

## 系統需求

- Windows 10 或 Windows 11
- .NET Framework 4.8
- Microsoft Edge（PDF 產生）
- Google Drive 備份需要網路及可用的預設系統瀏覽器
